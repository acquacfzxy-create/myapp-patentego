# Bundled Font Licenses

This app bundles selected Noto fonts to keep multilingual UI rendering consistent across Android and iOS devices.

Fonts included:

- Noto Sans CJK SC: Simplified Chinese UI text
- Noto Sans: Latin and Cyrillic fallback
- Noto Nastaliq Urdu: Urdu fallback
- Noto Sans Gurmukhi: Punjabi/Gurmukhi fallback

Sources:

- https://github.com/notofonts/noto-cjk
- https://github.com/notofonts/noto-fonts
