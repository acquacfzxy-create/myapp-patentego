## 2. 意大利语 (Codice della Strada) 语境校准

- **模拟考试**：全库意大利语已统一为 **「Simulazione Esame」**（非 "Test"），符合意大利《交通法》及驾考语境。
- 相关键已校准：`mock_test`、`mock_test_title`、`mock_test_completed`、`mock_exam_trend_title`、`no_mock_exam_records` 等。
- 及格线 / 错误上限：意语使用 **「Soglia (max 3 errori)」**，与 B 照最高错误数一致。

---

## 3. 简繁检查

- 所有中文词条均为 **简体中文**（如：需更多数据、暂无考试数据、当前考试通过率预测、预测精度：低、较昨天提升/下降、解锁深度分析、详细分析、及格线、重点攻克章节等）。

---

## 4. 视觉与长文本处理（俄语 / 乌克兰语）

- 以下位置已使用 **`FittedBox(fit: BoxFit.scaleDown)`**，避免长文案溢出：
  - 学习报告页「解锁深度分析」按钮文案；
  - 「重点攻克章节」标题；
  - 「模拟考试错误趋势」标题。
- 俄语、乌克兰语译文在可能较长的标题与按钮上会随空间自动缩放，避免 Overflow。

---

## 5. RTL（乌尔都语）与标点

- 乌尔都语为 RTL 语言；`MaterialApp.builder` 会按当前语言为 `ur` 时包一层 `Directionality.rtl`，确保页面级导航内容也翻转。
- 旁遮普语当前使用 Gurmukhi 字体，保持 LTR。
- 主页乌尔都语问候避免混入英文默认用户名，减少 BiDi 混排导致的标点错位。

---

## 6. 核心术语保护：Vero / Falso

- **「Vero」** 与 **「Falso」** 在任意语言下 **均不翻译**，始终使用意大利语原词。
- 实现方式：`question_widget.dart` 中答题按钮文案为硬编码 **`'VERO'`** 与 **`'FALSO'`**，未接入 `AppStrings`，保证与真实考试用语一致，便于用户适应正式考试。

## 9. 2026-06-01 本地化工程审查

### 9.1 本轮已修正

- 首次语言选择页不再维护页面内硬编码 switch，改为使用 `AppStrings.getWithLanguage(...)` 读取 `select_language` 与 `start_learning`。
- `MaterialApp.title` 不再使用中文硬编码，改为使用当前语言下的 `app_title`。
- 启动页语言预加载改为只接受 `UserState.supportedLanguages`，避免旧本地记录中的 `it` 被当作 UI 语言。
- RTL 方向只应用于乌尔都语 `ur`；当前旁遮普语使用 Gurmukhi 字体，保持 LTR。
- 错题复习解析缺失文案改用 `explanation_missing_short`，避免显示未定义的 `no_explanation` key。
- 为支持语言补齐一组关键发布路径文案：`start_learning`、`splash_slogan`、`no_image`、章节锁定标题/说明等。
- 继续补齐订阅、同步、登出删除云端数据、错题 VIP 引导、登录错误提示等剩余 key；当前 `it/zh/en/ru/ur/pa/uk` 均为 309 keys。
- 主页问候语与会员状态不再硬编码，改用 `home_greeting`、`premium_user`、`free_user`。
- RTL 方向控制移入 `MaterialApp.builder`，模拟器截图已验证乌尔都语首屏布局翻转。
- 新增测试守卫：支持的 UI 语言必须定义关键本地化 key；RTL 只能应用于乌尔都语。
