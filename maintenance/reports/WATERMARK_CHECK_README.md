# 图片水印检查与移除工具

## 功能说明

这两个 Python 脚本用于检查和批量移除题库图片中的水印。

### `check_watermark.py` - 水印检测脚本

**功能：**
1. 遍历 `images/img_sign/` 文件夹下的所有图片
2. 检测图片右下角 15% 区域是否存在非空白内容
3. 统计疑似带水印的图片数量
4. 随机抽样显示 10 张检测到的图片路径
5. 保存抽样图片的右下角区域到 `watermark_samples/` 目录（用于可视化确认）

**使用方法：**
```bash
python check_watermark.py
```

**输出：**
- 控制台显示检测统计信息
- 随机抽样 10 张疑似水印图片的路径
- 在 `watermark_samples/` 目录保存右下角区域样本

### `remove_watermark.py` - 批量移除水印脚本

**功能：**
1. 裁剪图片右下角 15% 区域
2. 用白色填充裁剪区域
3. 可选择覆盖原文件或保存到新目录
4. 支持自动备份原文件

**使用方法：**
```bash
python remove_watermark.py
```

**配置参数（在脚本中修改）：**
- `BACKUP_ORIGINAL = True`: 是否备份原文件（默认：是）
- `OUTPUT_DIR = None`: 输出目录（None 表示覆盖原文件，或指定新目录路径）

## 注意事项

⚠️ **重要提示：**

1. **备份原文件**：建议在运行 `remove_watermark.py` 前，先手动备份 `images/img_sign/` 目录
2. **测试运行**：可以先在少量图片上测试，确认效果后再批量处理
3. **检查样本**：务必查看 `watermark_samples/` 中的样本，确认检测准确性
4. **不可逆操作**：如果选择覆盖原文件，操作不可逆（除非有备份）

## 技术细节

### 检测算法

- **检测区域**：图片右下角 15% × 15% 区域
- **白色阈值**：RGB 值 >= 240 认为是白色
- **水印判断**：非白色像素占比 > 5% 认为疑似有水印

### 处理方式

- **裁剪区域**：右下角 15% × 15%
- **填充颜色**：白色 (255, 255, 255)

## 故障排除

### 问题：找不到图片文件

**解决**：检查 `ASSETS_DIR` 路径是否正确，确保指向 `images/img_sign/` 目录

### 问题：OpenCV 无法读取某些图片

**解决**：可能是图片格式问题，脚本会自动跳过并继续处理其他图片

### 问题：处理后的图片质量下降

**解决**：检查图片格式，确保使用无损格式（PNG）保存
